## Data and code for MW project associated with Chen, S., Stocco, A., & Yamasaki, B. L.
### Repository Structure
* analysis/: Contains scripts used to reproduce the analyses and statistical models reported in the project.

* data/: Contains preprocessed datasets used in the analyses.
  * MW_all.csv: Behavioral data for all trials.
  * MW.csv: Pre-probe behavioral data.
  * ddm_parameters/: Drift Diffusion Model (DDM) parameters estimated from the model, including parameters computed for both the full task and pre-probe analyses.
